HTML

  • HTML: The most basic building block of websites, it is used to create the structure of a webpage.
  • Html langeng: means that the language is set in english
  • Head: elements contain metadata that helps the browser to render the page correctly;not visible when a user looks at a webpage.
  • Styles: defines CSS styles
    • Base: defines base URL
  • Link: connects external resource to HTML
  • Meta: Defines metadata such as character set, description, keywords, authors, viewports
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • CSS: the language used to describe the presentation of a website created with HTML and how each element on a webpage is rendered
  • Box-shadow: This property adds shadow effects around an element's frame.
  • Border: property assigns a border with a width of 5px that is solid and gray
  • Margin: Property assigns 40px to the top and bottom, and the 'auto' value to the left and right
  • Card: a class selector, any element assigned to the class 'card' will gain these property assignments form the CSS

Git

  • Git status: checks what branch we are currently on
  • Git checkout -b 'branch name': creates a new branch and switches to it
  • Git Switch: Switches the branches
  • cb: takes you back all the way
  • Git add -A: Allows you to make a new directory

Javascript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • A function ends in parenthesis
